# HomeBrew-TrimUiSmart

<p>i'm trying to develop a homebrew for trimui smart for fun, not a serious project</p>

<h2>Requirements</h2>

```
    docker
    docker desktop
```

# how to run

```
docker compose up
# wait to build image
# exec
docker-compose exec desenvolvimento-arm bash
# to enter in setup
cd SOURCE_CODE
make
make run ARGS="arguments"
```
